# 泛用技能

## 挂钩属性：力量

:--:|:--:|:--:|:--:|:--:|:--:
15|威压|战技|被动|0|敌人通过一周围一圈的格子的时候，每通过1个格子都要消耗6点行动点
30|守护|战技|被动|0|可以代替周围1格的友军过防御检定；周围1格的友军也获得“威压”的效果

把对象举起来，从一个地方移动到另一个地方。

熟练度升级附赠技能：

30|人肉炸弹|特殊战技|力量|10|使用此技能若对抗成功，造成徒手攻击伤害的同时，可将对方举起来，并最多投掷差值因子等值距离远处，对投掷到的目标造成一半的伤害

### 缴械

目标生命值低于8点时，用拳头，或是武器的非致命部位攻击敌方，使敌方丧失战斗能力。

30|急速冲拳|特殊战技|力量|5|主武器为单手武器或没有武器时才能使用，突然攻击目标的要害，成功则对方抵抗判晕眩

## 挂钩属性：敏捷

### 闪避

战斗时常用的防守手段，看准敌人的攻击规律进行规避。

熟练度升级附赠技能：

熟练度等级|名称|战技/特殊战技|类型/挂钩属性|消耗SP|效果
:--:|:--:|:--:|:--:|:--:|:--:
15|同步|战技|被动|0|对方闪避成功时，你可以消耗1行动点过一个没有经验加成的闪避，如果过了，则视为对方闪避无效
30|多动|战技|被动|0|每闪避成功一次，获得1点额外的行动点，这些行动点会在你的下一次行动结束后消失

### 妙手

考验手部动作的灵敏程度的行动，可以用于偷盗、撬锁，或者其他精细操作。

熟练度升级附赠技能：

熟练度等级|名称|战技/特殊战技|类型/挂钩属性|消耗SP|效果
:--:|:--:|:--:|:--:|:--:|:--:
15|巧取|特殊战技|敏捷|5|战斗中，若你处于对方不可见的状态下，你可以用本技能偷取对方身上可见的可携带装备
30|圣手|战技|被动|0|除非大失败，你的一切妙手相关的行动不会引发注意，或是导致严重后果

### 潜行

放低姿态，融入环境，变得不容易被他人发现。

熟练度升级附赠技能：

熟练度等级|名称|战技/特殊战技|类型/挂钩属性|消耗SP|效果
:--:|:--:|:--:|:--:|:--:|:--:
15|遁影|战技|被动|0|在没有敌方角色发现的情况下，一个成功的潜行检定可以让你直接进入一种高级的潜行状态：隐身，持续1回合
30|息声|战技|被动|0|你以潜行或隐身状态进入敌人感知范围/4后，敌人不再获得自动识破隐形的机会，除非你做出敌意行动

!!! info "注：关于隐身和普通潜行"
    隐身：从所有角色的视线内消失，无法成为非AOE的攻击目标，目标对抗你的敌意行为时要恰一枚惩罚骰。

    识破隐形：若处于某角色的感知范围/4的范围内，或者对某目标做出敌意行动后，需要用潜行与目标的侦查或聆听进行对抗，若失败，则解除隐身状态。

    破隐一击：可以主动放弃隐身状态进行一次强有力的打击，这次敌意行动恰一枚奖励骰。

    幽灵隐身：若使用本技能的种族为幽灵，则同时获得穿越障碍物的特性。前提是穿越前需要知道障碍物的厚度。

    普通的潜行：在没有敌方角色发现的情况下，成功的潜行检定可以进入“潜行”状态：目标对抗你的敌意行为不需要额外恰惩罚骰，进行敌意行动后强制破影一击，其他与隐身没有区别。

### 侦查

通过细致入微的观察，可以发现远处的目标，或是近处的细节，非常常用的技能。

:--:|:--:|:--:|:--:|:--:|:--:
15|细致入微|战技|被动|0|以后侦查的大失败均视为普通失败；就算是普通失败，也会获得少量有价值的信息
30|洞悉一切|战技|被动|0|你可以直接知晓你视野范围内所有角色的属性值

### 感知
集中注意力到听觉、嗅觉等视觉以外的感官，可以察觉到环境中一些细微的骚动，非常常用的技能。

熟练度升级附赠技能：

:--:|:--:|:--:|:--:|:--:|:--:
15|快速反应|特殊战技|敏捷|0|被敌人攻击时使用，如果检定成功，则可以立刻改变自己的朝向，对抗隐形敌人时抵消惩罚骰
30|自走雷达|战技|被动|0|你可以直接知晓在场所有未知敌人与你之间的相对方位，且对抗直射时的闪避阈值不减半，识破隐形时获得一枚奖励骰

## 挂钩属性：体质

### 抵抗

泛指你的身体素质，可用于抵消debuff的附加，在致命伤的情况下坚持更久时间，或是抵抗你的睡意。

熟练度升级附赠技能：

熟练度等级|名称|战技/特殊战技|类型/挂钩属性|消耗SP|效果
:--:|:--:|:--:|:--:|:--:|:--:
15|重整态势|特殊战技|体质|0|无论成功与否，均可恢复50％的SP，代价是本回合受到伤害*1.5倍。若成功，则只需消耗基础行动点，否则需要消耗全回合行动点
30|呼吸回血法|战技|触发|0|可以用行动点来恢复HP。每1点行动点恢复5％的HP，每回合最多可以用此技能恢复50％HP

运用你在本世界的所见所闻得来的知识来鉴别某一事物，常用于估价、鉴定物品，或是判断某一客观说辞的真实性。

熟练度升级附赠技能：

熟练度等级|名称|战技/特殊战技|类型/挂钩属性|消耗SP|效果
:--:|:--:|:--:|:--:|:--:|:--:
15|见多识广|战技|被动|0|所有挂钩智力、魅力或魔力的检定获得-5奖励（不适用于护短技能）
30|护短|战技|被动|0|以检定值获得+15惩罚的代价，可用见闻检定代替任何一项与智力、魅力或魔力挂钩的检定
