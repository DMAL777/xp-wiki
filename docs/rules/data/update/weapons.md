# 武器熟练度

原则上来说，隶属于某种武器类型分支下的技能只能在持有该种类的武器时使用，不过如果画风合适，或者技能中有特殊说明，主持人也可以考虑允许跨武器种类使用。

熟练度等级|名称|战技/特殊战技|类型/挂钩属性|消耗SP|效果
:--:|:--:|:--:|:--:|:--:|:--:
5|稳固|战技|被动|0|巨盾系武器额外获得全身甲+1的特性，若主武器为单手武器，作为副武器时，无需切换武器也能享受特殊效果带来的增益
15|劈砸|特殊战技|与武器一致|8|若成功，武器威力减半，差值因子×1.5倍，让对方抵抗判<a href="../../status/normal/#晕眩" target="_blank">“晕眩”</a>
25|盾墙|战技|触发|8|消耗4点行动点将巨盾插在地面上原地固守，立即结束你的回合，到下次你的回合开始为止，处于左右和后方格子的友军和你完全免疫来自正面和侧面远程伤害

巨盾熟练度达到10、20和30级时，无法学习额外的通用武器升级技能，但可以选择升级战术能力中“架势·坚守姿态”或“背靠背”的阶级。这种升级仅在主武器为“巨盾系”的情况下才会生效，学会巨盾5级升级技能“稳固”后，巨盾系作为副武器时这种升级也会生效。
