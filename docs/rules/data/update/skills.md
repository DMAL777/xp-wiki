### 潜行

放低姿态，融入环境，变得不容易被他人发现。

熟练度升级附赠技能：

熟练度等级|名称|战技/特殊战技|类型/挂钩属性|消耗SP|效果
:--:|:--:|:--:|:--:|:--:|:--:
15|遁影|战技|被动|0|在没有敌方角色发现的情况下，一个成功的潜行检定可以让你直接进入一种高级的潜行状态：隐身，持续1回合
30|息声|战技|被动|0|你以潜行或隐身状态进入敌人感知范围/4后，敌人不再获得自动识破隐形的机会，除非你做出敌意行动

!!! info "注：关于隐身和普通潜行"
    隐身：从所有角色的视线内消失，无法成为非AOE的攻击目标，目标对抗你的敌意行为时要恰一枚惩罚骰。

    识破隐形：若处于某角色的感知范围/4的范围内，或者对某目标做出敌意行动后，需要用潜行与目标的侦查或聆听进行对抗，若失败，则解除隐身状态。

    破隐一击：可以主动放弃隐身状态进行一次强有力的打击，这次敌意行动恰一枚奖励骰。

    幽灵隐身：若使用本技能的种族为幽灵，则同时获得穿越障碍物的特性。前提是穿越前需要知道障碍物的厚度。

    普通的潜行：在没有敌方角色发现的情况下，成功的潜行检定可以进入“潜行”状态：目标对抗你的敌意行为不需要额外恰惩罚骰，进行敌意行动后强制破影一击，其他与隐身没有区别。
