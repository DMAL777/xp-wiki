# 修士/修女

## 概述

修士（男性）或者修女（女性）是皈依天教的信徒，在圣堂中负责最基本的基层工作，内容包括祈祷与抄写经文。

## 必要条件

* 信仰天教

## 关键属性

* 魔力

## 熟练技能

* 魔法感知
* 魔法抗性
  
## 职业特性

获得临时特殊技艺：抄写经文。
