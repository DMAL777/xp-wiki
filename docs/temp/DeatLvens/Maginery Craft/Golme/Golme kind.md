# 魔导造物制造需求及其平均性能

魔导造物属于*BOSS/玩家类型单位，其力量、敏捷、体质、魅力属性由组成的全部零件总和构成，智力与魔力则由绑定的“主人”的基础值与成长值（如果有）以及特殊模块决定。

同时，魔导造物的HP与SP将不与属性绑定，单独由零件与魔导核心提供。
