    自然类魔法

    丛林系魔法

    绿色是健康的颜色，但如果你在一片绿色中见到一个丛林系法师，那就不一定了

    阶级 名称 流派 SP消耗 初始效果 10级进阶效果 20级进阶效果 30级进阶效果
    初级 飞叶快刀→落叶龙卷 旋涡之法 8 造成魔法威力为3的伤害，增益：可消耗自身所处丛林地形使造成的伤害*1.5倍 魔法威力变为5 魔法威力变为8 可以选择让效果范围扩大最多2格，但增益消耗的草地地形也要同步扩大最多2格
    初级 植树术→造林术 心神之法 0 目标位置生成一颗幼树，可成长；若目标位置为丛林地形，则幼树自动成长 可以免费在目标位置周围1格再生成一颗幼树 可以免费在另外一个位置生成一颗幼树 可抛弃10级和20级的效果，选择33~55的范围批量生成幼树
    中级 缠绕术→藤蔓术 心神之法 0 目标附加1回合“缠绕”状态；增益：若目标处在丛林地形，可消耗地形延长持续时间1回合 缠绕状态下每回合-3HP，增益：若目标处在丛林地形，可消耗地形再-3HP 缠绕状态下所有行动消耗1.5倍行动点 在3*3的范围内生成藤蔓1回合，通过藤蔓的角色若抵抗没过，自动陷入缠绕状态
    中级 生草术→野蛮生长 心神之法 0 在3*3的范围内生成人造丛林地形 范围可扩展为4*4 可以在允许范围内自由选择生成人造丛林地形的格子 可以选择一片人造或自然丛林地形，随后在其最外层扩展1格人造自然地形
    上级 绿色沼泽→绿色风暴 旋涡之法 12 选择一片人造或自然丛林地形，对其上的所有敌军进行一次无增益的飞叶快刀和缠绕术 增益飞叶快刀效果，不消耗地形 增益缠绕术效果，不消耗地形 可选择消耗这一大片地形，来让最终效果翻倍

    山岳系魔法

    崎岖的山路会让人难以攀爬，但对于山岳系法师来说却如履平地

    阶级 名称 流派 SP消耗 初始效果 10级进阶效果 20级进阶效果 30级进阶效果
    初级 势能冲击→势能波 旋涡之法 8 使用者需接地，造成魔法威力为（高度差）的伤害，并可选择击退对方（高度差/2）格 魔法威力变为（高度差*1.5） 魔法威力变为（高度差*2） 除了目标外，还可以对水平距离在施法范围以内的所有接地的敌方造成减半的效果
    初级 弹跃术→山羊脚 心神之法 0 目标附加1回合“弹跃”状态 跳高最多可跳到3米，计算自由落体伤害时无视3米高度 跳高最多可跳到4米，计算自由落体伤害时无视4米高度 对于一个攀爬对象，只要存在坡度，即可在上面自由活动，无需再过攀爬判定
    中级 天降术→死从天降 心神之法 0 目标附加1回合“天降”状态 最高有效高度差变为4 最高有效高度差变为5 天降时免费赠送一次近战武器攻击
    中级 坡度制造→山岳制造 心神之法 0 令目标作为中心点（要求为平地），从中心点隆起，生成最多1层高坡 最多2层高坡 最多3层高坡 可以在高坡影响的范围内自由选择隆起的高度，最多为3
    上级 地震术→地动山摇 旋涡之法 12 在4*4范围内产生地震，范围内所有角色受到魔法威力为5的伤害，抵抗判倒地 从范围最外圈开始塌陷，每往内缩一圈，高度-1 范围可扩展到5*5，魔法威力变为8 倒地效果变为晕眩

    川流系魔法

    清澈的水流令人舒心，但也伴随着危险，而川流系魔法则可以让它们变得无害，或者更加致命

    阶级 名称 流派 SP消耗 初始效果 10级进阶效果 20级进阶效果 30级进阶效果
    初级 水下呼吸→空游自在 心神之法 0 目标附加1回合“水下呼吸”状态 持续时间变为2回合 水下行动行动点消耗从2倍变为1.5倍 水下行动与陆上行动无异
    初级 踏浪术→冲浪术 心神之法 0 目标附加1回合“踏浪”状态 持续时间变为2回合 顺浪而下时，冲锋类型的技能获得一枚奖励骰，且经过的路径不消耗额外行动点 顺浪而下之后的第一个攻击检定有-10奖励
    中级 弄潮术→驭波术 旋涡之法 12 消耗川雾地形，对直线范围内所有角色造成魔法威力5的水系伤害，路径上产生“浪潮”地形 魔法威力变为8 浪潮地形可以持续2回合 可选择让目标直线范围往左右两旁扩展出2条
    中级 迷雾术→雾隐术 心神之法 0 在最多5*5范围内产生1回合“迷雾”地形 范围可扩展到7*7 持续时间延长1回合 生成的“迷雾”地形中的敌方单位抵抗判定致盲状态
    上级 窒息水汽→致命水汽 心神之法 0 选择一片川流或迷雾或水洼地形，之后在该片区域内的所有角色每回合都要进行“溺水”检定 现在可以手动取消窒息水汽 溺水同时造成等量的SP伤害 进行溺水检定时需要消耗2行动点

    大地系魔法

    只要脚踏实地，就是大地系魔法的天下

    阶级 名称 流派 SP消耗 初始效果 10级进阶效果 20级进阶效果 30级进阶效果
    初级 冲击波→震荡波 旋涡之法 8 对直线范围内所有角色造成魔法威力3的接地伤害 魔法威力变为5 魔法威力变为8 可选择让目标直线范围往左右两旁扩展出2条
    初级 大地加护→大地馈赠 心神之法 0 目标附加1回合“大地加护”状态 增加效果：进攻检定值有-5奖励 增加效果：无法被位移 增加效果：获得额外2点行动点，之前所有的效果倍增
    中级 造路术→铺路术 心神之法 0 选择范围内最多3格变为通路地形1回合（无法消除障碍物） 最多5格 持续时间延迟1回合 可以选择让目标格周围一圈也变为通路地形
    中级 撼地术→战争践踏 旋涡之法 10 对周围2格范围内的所有敌方角色造成魔法威力为3的接地伤害，对方抵抗判眩晕 魔法威力变为5 范围扩展为周围3格 可以让作用范围内的特殊地形变为通路地形1回合

    特效名称 特效效果
    人造丛林地形 通过这种地形需要消耗1.5倍的行动点；处于这种地形上的角色，潜行检定时获得一枚奖励骰；在这种地形的角色会受到1.5倍的火焰伤害，并且会在原地产生5点固定火伤的火墙，每回合这种火墙会向外扩散一格，2回合后熄灭，并把这种地形变成通路地形；气系魔法可以使作用目标格的丛林地形在这场战斗中变为通路地形
    自然丛林地形 不同于人造丛林地形，自然丛林地形不会因为消耗或者燃烧而变成通路地形，它会一直存在，而且如果你尝试火烧自然丛林地形，可能会遭到自然神灵的报复
    幼树 相当于有半个人高度的半身掩体，拥有5+（差值因子*3）生命值，受到攻击时对抗阈值为20
    成长 再次对幼树使用植树术的话，将幼树成长为成树
    成树 相当于有1.5个人高度的全身掩体，在幼树的基础上增加5+（差值因子*3）生命值，受到攻击时对抗阈值为50
    缠绕 无法移动，无法闪避（阈值固定为10）
    自由落体伤害规则 当从高处自由下落时，每高出的1米会在下落时提供1d5点损伤。可以进行跳跃检定，若成功，则造成损伤减半。若下方有角色，则下方的角色会受到下落者同样的伤害，并附加下落者体质/5的额外伤害
    弹跃 攀爬、跳跃检定获得一枚奖励骰，且跳跃（消耗2行动点）可以跳高到最多2米的高度，计算自由落体伤害时无视2米高度
    天降 "下落过程中，计算下落的高度差，对落地点及其周围（高度差）格的所有接地的敌方造成（高度差）d5点物理伤害，最高有效高度差为3（注：浮空术会抵消掉天降的效果，两者可以同时存在，但生效时你只能选择一种效果；兼容自由落体伤害原则"）
    高坡 中心点最高，周围的高度依次降低，直到原本的高度（例如1层高坡中心点高度+1，周围1格为原来高度；2层高坡中心店高度+2，周围1格高度+1，最外圈为原来高度；以此类推）；从高坡上往下使用的冲锋类技能时获得一枚奖励骰，且经过的路径消耗的额外行动点减半；从高坡上往下射击，射程无视高度部分，检定获得-5奖励；从高坡往下走，行动点消耗减半；从下往高坡上走，行动点消耗*1.5倍
    倒地 任何对抗阈值减半，需要花费2点基础行动点解除；此外，如果倒在水洼里，会附加潮湿状态
    川流地形 若为浅水区，通过这种地形需要消耗双倍的行动点；若为深水区，则适用水下战斗原则；处于这种地形上的角色受到的雷电伤害减少75％；处于这种地形上的角色不会受到火焰伤害，且也无法制造出火焰；处于这种地形上的角色使用水系魔法时获得一枚奖励骰
    迷雾地形 角色可以暂时消除周围1~3格的迷雾地形（仅对自己有效），但你也可以通过侦查或聆听检定来获取4~8格内迷雾地形中的场景或角色信息；处于这种地形上的角色使用气系和水系魔法时获得一枚奖励骰；迷雾地形上的导电效果无视导电距离（类似水洼），且跳数+1
    弄潮术川雾地形消耗 选择迷雾或水洼或川流地形作为目标，接着在选择的目标地点选择方向，魔法从目标地点发出。若地形为迷雾，则消耗时，本地形所处的一大片迷雾或水洼地形会向内缩一圈；若地形为水洼，则消耗该地形；若地形为川流，无论深浅，均可无限次消耗
    水下呼吸 水下行动时，不再需要进行“溺水”检定；若原来已有“溺水”状态，则立即解除
    踏浪 允许在水上行走，可免行动点在“浪潮”地形正方向上行走（顺浪而下），且不会被“浪潮”地形强制冲走
    浪潮地形 在浪潮地形上的角色强制潮湿，需要过一个抵抗，否则强制消耗行动点往浪潮正方向移动2格；往浪潮正方向移动，消耗行动点减半；往浪潮反方向移动，消耗行动点翻倍；浪潮地形会在一回合后消失
    大地加护 只在目标接地的情况下生效：获得全身10％减伤
    通路地形 白板地形，没有任何效果
    接地伤害 只对接地的敌人产生效果，可跨障碍物和高度进行打击
    昏暗地形 你无法得知昏暗中的场景或角色信息，但有些信息可以通过聆听检定来获得；有火焰附着的道具（如火焰附魔武器或火把）可以暂时消除周围1~6格的昏暗地形；处于这种地形上的角色所有与视觉相关的检定获得一枚惩罚骰
    黑暗视觉 无视昏暗地形的惩罚
    光亮术 可以对物品使用；目标在持续范围内提供1~6格的光照
    晕眩 无法行动，任何对抗阈值全部变为10，需要花费6点基础行动点，或者受击解除
    致盲 武器范围和施法范围的最大值变为2，所有与视觉相关的行动检定前要先过聆听，若没过，则掐一枚惩罚骰
    自然荒漠地形 通过这种地形需要消耗1.5倍的行动点；若某角色在此地形上停留1回合以上，附加“膈应”状态，所有检定值+5惩罚，需要花费2行动点解除；处于这种地形上的角色使用火系魔法与土系魔法时获得一枚奖励骰，使用水系魔法时获得一枚惩罚骰；可被无限消耗
    人造荒漠地形 通过这种地形需要消耗1.5倍的行动点；若某角色在此地形上停留1回合以上，附加“膈应”状态，所有检定值+5惩罚，需要花费2行动点解除；处于这种地形上的角色使用土系魔法时获得一枚奖励骰；被消耗时会变回通路地形
    瘸脚 移动消耗的行动点变为2倍，与脚有关的行动检定获得一枚惩罚骰
    致盲 武器范围和施法范围的最大值变为2，所有与视觉相关的行动检定前要先过聆听，若没过，则掐一枚惩罚骰
    流沙陷阱 使第一个经过的敌人过抵抗，若抵抗失败，则附加“瘸脚”状态，然后陷阱消失；陷阱对敌人不可见，敌人移动到近旁时需要过个侦查，若过了，才能发现陷阱
    沙行 无视荒漠地形一切负面效果；无视流沙陷阱的效果；在自然荒漠地形中移动行动点消耗减半；可消耗脚底下的荒漠地形来获得全身10％减伤，持续1回合
    沙尘暴 对经过角色造成魔法威力为8的岩土伤害，附加一回合“致盲”状态
