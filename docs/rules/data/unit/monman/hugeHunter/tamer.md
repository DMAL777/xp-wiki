# 驯兽师

## 形象

暂缺

## 种族（不会应用其种族特性）

鬼族、高阶僵尸、高阶骷髅、其他下层世界生物

## 简介

善于调教巨兽的巨兽猎兵会被擢升为驯兽师。相当重要的后续清场、收集战利品，以及巨兽经济的运营工作也由他们来兼职。

## 活动范围

驯兽师通常会跟随他们的老大，也就是隶属于巨兽猎手的代理头目一同行动，通常不作为战斗角色登场——除非他们骑上了野兽。如果你在市场上遇到了下层世界装扮的驯兽师，那估计附近会有巨兽猎手小分队驻扎。
