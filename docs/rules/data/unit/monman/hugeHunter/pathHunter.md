# 追迹猎手

## 形象

暂缺

## 种族（不会应用其种族特性）

鬼族、高阶僵尸、高阶骷髅、其他下层世界生物

## 简介

精通野生巨兽习性的巨兽猎兵会被擢升为追迹猎手。定位目标、布置陷阱是他们的拿手好戏。

## 活动范围

追迹猎手通常会跟随他们的老大，也就是隶属于巨兽猎手的代理头目一同行动，作为远程射手或陷阱师登场。除了与巨兽猎手的遭遇场合，你也会在巨兽可能出没的地区发现他们的踪迹。
