# 桐坂众散兵

## 形象

暂缺

## 种族（不会应用其种族特性）

鸦天狗（最多）、狐天狗、鲸天狗

## 简介

桐坂众散兵负责游走于战场前线，骚扰敌人的同时尽可能地保护后方的狙击手和战勤。因此，散兵通常在配备火器的同时精通一门近战武器，大多数散兵会直接选择远近一体的机巧杖，信奉斗争大神。

## 活动范围

桐坂散兵、狙击手和战勤基本以小队的形式活动。像桐坂众这样的佣兵集团，原本在天狗国各处接活。现在业务被迫扩展到彼岸，在哪看见他们都不会奇怪了。
