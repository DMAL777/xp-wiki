# 低阶僵尸

## 形象

暂缺

## 简介

被施法者从土地里拉出来的尚未腐烂的行尸被称为无魂僵尸，或是低阶僵尸。这种僵尸只能由施法者手动进行操控，没有任何自主行动能力。
