    通用武器技能

    原则上来说，隶属于某种武器类型分支下的技能只能在持有该种类的武器时使用，不过如果画风合适，或者技能中有特殊说明，主持人也可以考虑允许跨武器种类使用。

    远程武器

    初级

    名称 战技/特殊战技 类型/挂钩属性 消耗SP 效果
    近程射击 战技 被动 0 若目标在攻击范围/2内，进攻检定有-10奖励，且远程狙击无需额外消耗行动点
    援护射击 战技 被动 0 你的远程武器也可以提供夹击判定，并享受夹击效果
    物尽其用 战技 被动 0 搜刮/回收弹药自动成功；若使用防御来应对对方的远程攻击，则可直接回收对方的弹药
    机动 战技 被动 0 受到近战攻击后，立刻获得2行动力的移动机会，无视借机
    精准射击 特殊战技 与武器一致 5 进行一次只消耗1行动点的远程狙击；无视近战接触惩罚
    寒芒 特殊战技 与武器一致 5 若成功，造成伤害的同时，对方在击中的部位进行一次“伤口”抵抗检定，持续2回合

    伤口状态

    任何针对该部位的异常状态必定附加；任何针对该部位的攻击会造成一次破绽

    中级

    名称 战技/特殊战技 类型/挂钩属性 消耗SP 效果
    神射手 战技 被动 0 命中头部/弱点造成的最终伤害变为1.5倍，且造成一次破绽
    效力强化 战技 被动 0 任何弹药附加的异常状态可以享受以下效果二选一：持续时间翻倍，或者异常状态效果翻倍
    开门见山 战技 被动 0 每回合开始的第一次射击不消耗行动点，不算衰减
    凝神 战技 被动 0 每次射击可额外消耗n行动点使射击检定阈值+5n，最终伤害+n(n<=6)
    狙击武器 特殊战技 与武器一致 8 击中部位固定为左右手，若成功，造成伤害的同时，可以打落对方的武器，任何人可以花费2行动点捡起
    卸其羽翼 特殊战技 与武器一致 8 若成功，造成伤害的同时，强制造成一次破绽；若目标是空中单位，则造成1.5倍伤害并将其击落

    上级

    名称 战技/特殊战技 类型/挂钩属性 消耗SP 效果
    最佳角度 战技 被动 0 射程+0.5倍，无视射程中的阻碍（包括角色），但是没法穿墙（指无法穿过2个连续相邻的障碍物）
    暗袭 战技 被动 0 在对手措手不及的情况下进行射击，可以选择：本次攻击对方对抗阈值固定为5，但对方对你这次攻击的识破隐形检定获得一枚奖励骰
    流星雨 战技 被动 0 5次射击内，对于同一目标，每次射击结束后进行统计，任何部位击中次数超过3次，则该部位强制致伤
    完美谢幕 战技 被动 0 若本回合进行了3次及以上次攻击，则最后一次攻击不算衰减，检定值-10，最终伤害*1.5倍
    贯穿射击 特殊战技 与武器一致 10 攻击范围内一直线上的所有敌人，第一个敌人造成2倍伤害，之后每贯穿一个敌人伤害-0.25倍
    全神贯注 特殊战技 与武器一致 10 若成功则造成伤害，且此伤害为固伤，不会受到任何减免影响

    措手不及状态

    措手不及状态是一个相对状态。假设A进攻B，且A处于潜行、隐身、或其他B未察觉其进攻意图的情况下进攻B，那么此时B相对于A就是措手不及状态，此时B应对A进攻的防守检定就会获得一枚惩罚骰。

    什么是未察觉其进攻意图的情况？一个典型的例子就是B对A没有敌意，A却突然对B发动攻击。一旦A这么做，B就会立刻对A产生敌意，其对A的措手不及状态就会立刻解除。如果A能在B反应过来之前攻击到B，那么A就能吃到B措手不及带来的增益。

    短程武器

    初级

    名称 战技/特殊战技 类型/挂钩属性 消耗SP 效果
    寸短寸险 战技 被动 0 目标正面朝你时，检定阈值+5，检定值-5
    顺水推舟 战技 被动 0 受到近战攻击时，如果对抗成功，造成3点固定伤害，若对方体型与自己相同，则可以改变对方的朝向或者交换双方的位置
    协调 战技 被动 0 短程武器作为副手武器时，若可双持进攻，主副武器的武器威力不再减半
    不绝连击 战技 被动 0 使用短程武器进攻时，攻击阈值衰减速度减半
    断筋击 特殊战技 与武器一致 8 进行一次瞄准对方肌腱的攻击，命中固定为腿部。若造成伤害，则对方抵抗判断筋1回合
    迅击 特殊战技 与武器一致 0 进行一次不算衰减的进攻

    断筋状态

    每移动一格受到移动所耗行动点两倍的伤害

    名称 战技/特殊战技 类型/挂钩属性 消耗SP 效果
    偷袭 战技 被动 0 在对手措手不及的情况下进行的攻击伤害翻倍
    卧薪尝胆 战技 被动 0 受到敌人攻击时，将攻击造成的伤害累积，下一次攻击最终伤害+（累积伤害*2）％
    趁虚而入 战技 被动 0 使用短程武器进攻陷入异常状态的目标时，进攻检定获得一枚奖励骰
    打带跑 战技 被动 0 若攻击成功，获得2点仅可用于移动的额外行动点，最多可以获得8点，一回合后失效
    连打 特殊战技 与武器一致 10 投掷2d2决定攻击次数，随后用此技能进攻，造成的伤害乘上攻击次数
    残废击 特殊战技 与武器一致 8 进行一次致残打击，命中固定为腿部。若造成伤害，则对方抵抗判瘸脚1回合

    瘸脚状态

    移动消耗的行动点变为2倍，与脚有关的行动检定获得一枚惩罚骰

    上级

    名称 战技/特殊战技 类型/挂钩属性 消耗SP 效果
    平步青云 战技 被动 0 无视一切对移动带来的不利影响（但不能穿墙）；在地面上移动消耗行动点减半
    娴熟打击 战技 被动 0 使用短程武器，武器威力附加自身敏捷/10（四舍五入）
    暗杀 战技 被动 0 在对手措手不及的情况下进行的攻击，若成功，则额外造成100％/40％/20％（小兵/精英/Boss）HP的固伤。每场战斗每人限定一次
    灵活走位 战技 被动 0 脱离交战状态现在不会造成破绽，无视威压
    关节打击 特殊战技 与武器一致 8 无论成功与否，都会对对方造成10点精神力损伤和2点行动力损失；若成功，附加8点固定伤害
    点穴 特殊战技 与武器一致 10 进行一次1.5倍伤害的攻击，根据击中部位让对方以一半的阈值抵抗判致伤（头部除外），该致伤效果只在1回合内有效

    中程武器

    初级

    名称 战技/特殊战技 类型/挂钩属性 消耗SP 效果
    二刀流 战技 被动 0 任何武器作为副手武器时，若可双持进攻，副武器进攻阈值不再减半；若以远程武器作为副手武器，则无视近战接触惩罚
    双手共持 战技 被动 0 将单手武器视为双手武器，武器威力附加（力量/10四舍五入)，但所有进攻需要消耗的行动点+1。可消耗1行动点来激活/取消此被动
    精准格挡 特殊战技 被动 0 每回合一次，对抗对方主动近战攻击时，如果对方阈值差值减去你的阈值差值小于等于该技能等级，则完全抵消本次伤害。
    每被动防守一次都可使本技能+2经验，上限30级
    顺势 特殊战技 被动 0 每回合一次，主动攻击敌人时，如果对方阈值差值减去你的阈值差值小于等于该技能等级，则本次攻击仍可无视对方的防守手段。
    每次主动攻击都可使本技能+2经验，上限30级
    重劈 特殊战技 力量 5 进行一次检定值+10惩罚的攻击，若击中，以下效果二选一：伤害*1.5倍，或让对方判定晕眩一回合
    弓步突刺 特殊战技 敏捷 5 可以攻击到直线上攻击范围x2的敌人，检定值-5，前提是中间没有障碍物

    中级

    名称 战技/特殊战技 类型/挂钩属性 消耗SP 效果
    单刀进枪 战技 被动 0 被射程大于自己的近战武器攻击时，可在攻击结束后免行动力前进到使对方处于自己武器射程最外圈的位置，且不会露出破绽
    疾走强袭 战技 被动 0 如果使用冲锋技能移动到了目标一格范围内发动攻击，则造成1.5倍伤害，且无视对方护甲值（减伤仍然有效）
    杀人剑 战技 被动 0 每次进攻可以以消耗自身10％的HP为代价，使得本次进攻检定获得-10奖励
    活人剑 战技 被动 0 攻击造成伤害后，可以以牺牲一半的伤害为代价，将这一半伤害作为恢复平分给自己的HP和SP
    要害打击 特殊战技 与武器一致 5 对于同一目标，第一次使用本技能，检定值-25，之后每次使用本技能，只能让检定值-5
    连续斩 特殊战技 与武器一致 8 若成功，造成伤害的同时，触发“连续斩”派生机制后，对自己叠加一层“连续斩”状态，最多叠加3层

    连续斩状态派生机制

    持续整场战斗。在使用特殊战技“连续斩”成功后触发，若“连续斩”层数大于等于1，则再进行一次连续斩对抗（不消耗行动点），将造成的伤害乘上连续斩的层数。
