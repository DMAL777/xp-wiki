# 巨兽猎兵

## 形象

暂缺

## 种族（不会应用其种族特性）

鬼族、高阶僵尸、高阶骷髅、其他下层世界生物

## 简介

巨兽猎兵是巨兽猎手的基本构成，所有巨兽猎手成员都必须从猎兵做起，进行针对巨兽的基本训练。

## 活动范围

巨兽猎兵通常会跟随他们的老大，也就是隶属于巨兽猎手的代理头目一同行动，充当一线士兵。你通常会在与巨兽猎手的遭遇场合中遇到他们。
