# 活物系魔法

## 活物系魔法共通效果

### 活物系的召唤物特性

* 均为小兵单位，1次可以召唤若干个，额外投入塑型土效果对这若干个都有效
* 所有技能的基础阈值都是召唤者对应召唤魔法的阈值
* 所有优势检定获得一枚奖励骰，所有劣势检定阈值-20
* 不受任何话术影响，无论正面或负面
* 可被炼金类法术中的修复术、塑型土修复术和消解术影响，可被急救影响

### 特殊技艺：活物系精通

* 初始：允许消耗3SP使用本特殊技艺（算作魔法），选定你已经学会的活物系召唤物，消耗其十分之一的塑型土，制作一个该召唤物类型的迷你玩偶
* 10级进阶：在召唤活物系召唤物时，检定获得一枚奖励骰
* 20级进阶：所有活物系召唤物额外投入塑型土上限变为：（召唤时塑型土消耗×0.75+施法者经历点×10）g
* 30级进阶：召唤活物系召唤物时，消耗塑型土减少20％

## 初级魔法

### 召唤影狼

高机动性的战士，擅长切入敌人后排，造成大量伤害。缺点是较为脆弱，只要稍微有一些武装就能把它们击退。

#### 魔法流派与消耗

* 塑型之法
* SP消耗：5
* 塑型土消耗：300g
* 模块指令槽：2
* 激活行动点：2

额外投入塑型土效果：

* 最多总共额外投入（150+施法者经历点×10）g
* 每多投入10g，HP、SP+1（最多投入150g）
* 每多投入20g，进攻无视1点护甲值（最多投入40g）
* 每多投入30g，提升“冲锋”或“夹击”或“强袭姿态”1个战术等级
* 投入50g，获得隐形效果
* 投入80g，获得影狼爪击效果
* 投入80g，获得战术15级能力
* 投入100g，获得影步效果

特殊技能效果：

* 隐形：花费5点SP，直接进入隐身状态
* 影狼爪击：攻击时消耗5点SP触发，造成1.5倍伤害，对方抵抗判<a href="../../../status/normal/#流血" target="_blank">“流血”</a>
* 影步：移动无视威压

#### 效果

* 初始：在施法范围内召唤3个影狼
* 10级进阶：塑型土消耗减少10％，塑型土进攻无视1点护甲值增益上限+20
* 20级进阶：塑型土消耗再减少10％，塑型土进攻无视1点护甲值增益上限+20
* 30级进阶：优势检定中增加“进攻”，塑型土进攻无视1点护甲值增益上限+20

#### 影狼属性

* HP：15
* SP：20
* 优势检定：无
* 劣势检定：防御、魔法抗性
* 武器威力6，攻击范围1
* 冲锋、夹击、强袭姿态（可用）初级，其余原始
* 没有护甲

## 中级魔法

### 召唤战猪

模拟野猪的召唤物，具有相当的攻击性，且能够对进攻的目标造成位移。防御能力较差，但足够坚持到完成它们的进攻任务。

#### 魔法流派与消耗

* 塑型之法
* SP消耗：8
* 塑型土消耗：550g
* 模块指令槽：3
* 激活行动点：3

额外投入塑型土效果：

* 最多总共额外投入（275+施法者经历点×10）g
* 每多投入10g，HP+1（最多投入150g）
* 每多投入20g，进攻无视1点护甲值（最多投入40g）
* 每多投入20g，护甲值+1（最多投入40g）
* 每多投入30g，提升“冲锋”或“夹击”或“侧翼攻袭”1个战术等级
* 投入50g，获得挑飞效果
* 投入80g，获得野蛮冲撞效果
* 投入80g，获得战术15级能力
* 投入100g，获得暴怒效果

特殊技能效果：

* 挑飞：每次成功的进攻可选择将目标挑飞差值因子格
* 野蛮冲撞：冲锋过程中，与第一个敌人对抗成功可以推着对方继续前进，推到截停物（如障碍物或第二个敌人）后造成目标与截停物5点固伤并停止
