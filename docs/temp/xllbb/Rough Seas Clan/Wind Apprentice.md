# 气浪学徒

## 概述

？？？（未解禁内容）

## 必要条件

* 种族为开智海族或人鱼族
* 你的魔力总值＞＝30
* 未拥有其他学徒职业
* 信仰‘风主’

## 关键属性

* 魔力

## 熟练技能
 
* 魔法抗性

## 职业特性

造成灵气伤害时，恢复自身造成伤害一半的Hp（单次恢复上限5）。
