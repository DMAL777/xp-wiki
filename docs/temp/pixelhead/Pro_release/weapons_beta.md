# 武器及其平均性能表

## 升级公式

每升一级可以选择让：

* 力量阈值-5（最低5，最高60）
* 武器威力+1（最多+10，最低-4）
* 持有“有属性加成”标签的武器，获得2点自由分配的装备属性加成（每种属性最多+6，最低-6）
* 近战武器无法加射程；若为远程武器，获得1点射程（最高+5，最低-5）
* 没有施法范围的武器（默认1施法范围），可以花费3等级获得1施法范围加成，此后每一级可以获得1施法范围加成（最高+8，最低-8）

每件武器上限15级，每提升1级都会让提升等级的代价增大，且不能降级

不过，你可以通过反向操作等量的上述值来把好处匀给其他属性，代价是消耗量变为2倍（比如通过增加10点力量阈值来让武器威力+1）

此外，武器在升至5、7、9、11级时会迎来质变，这会分别为武器提供一个初级、中级、上级、特级印痕位并同时印刻对应的印痕，而当等级到达13与15时，更是会进一步激发武器的潜能，分别获得一个初级与一个上级、一个中级与一个特级印痕位。特别的，你也可以选择将一个高级的印痕位拆分成两个低一级的印痕位并印刻，但不能反向操作且同种印痕无法超过三个

质变武器会为使用者带来多样化的好处，但质变后的武器使用难度更高（若武器熟练度低于对应武器等级的要求，则使用时阈值减半），同样的，也只有那些技艺精湛的铁匠才能开启高等级印痕位并印刻

重铸武器需要在相应质变等级所需求的铁匠处才能重铸，且每次只能将至多一个印痕转化成同级别的另一个，若要转化更多的印痕或是拆分印痕位则需要对其中的每个额外支付重铸价格的一半

## 铁匠铺印痕效果表

铁匠铺只能印刻有限的几种印痕，不过如果有新的未知印痕他们也很乐意为你解析并印刻————只需要付出重铸价格即可

初级|争锋刻痕|使用该武器进攻面对你的目标时，进攻阈值+5|无
中级|精进刻痕|获得“有属性加成”的标签|至少有三项属性达到50
中级|无羁刻痕|该武器已印刻冲劲刻痕的话，使用该武器进攻且对抗成功时，因子倍率+0.3且威力+1|该武器已印刻冲劲刻痕且尚未印刻无羁刻痕
中级|奋发刻痕|使用该武器进攻且对抗失败时，若对方奖励骰数量多于你，则下次进攻获得一枚奖励骰|该武器尚未印刻奋发刻痕
中级|破坚刻痕|使用该武器进攻时，瞄准部位的攻击对护甲生命值造成的伤害无视至多2点护甲值|该武器尚未印刻破坚刻痕
上级|突破刻痕|检定时，可以以令该武器提供的正面属性加成在战斗结束前合计下降2为代价，令当次检定阈值+10，检定值-5|拥有“有属性加成”的标签
上级|迅捷刻痕|每回合一次，使用该武器进攻时，行动点消耗-1且不算衰减|该武器尚未印刻迅捷刻痕
上级|变通刻痕|获得时指定一个魔力以外的属性，该武器也可挂那个属性使用|某一属性比武器原本挂钩的属性至少高15
上级|断钢刻痕|该武器已印刻破坚刻痕的话，使用该武器进攻时，瞄准部位的攻击再无视至多2点护甲值|该武器已印刻破坚刻痕
特级|智识刻痕|该武器已印刻求知刻痕的话，进攻结束后可以以将那次获得的经验降为0为代价，使下次进攻检定获得相当于（经验下降量×2）的阈值奖励，若该效果提供的阈值奖励大于等于8，则再获得一枚奖励骰|该武器已印刻求知刻痕且尚未印刻智识刻痕
特级|猛袭刻痕|该武器已印刻冲劲刻痕及无羁刻痕的话，使用该武器进攻时，可以以进攻结束时额外叠加5点进攻阈值衰减为代价，令当次阈值+10，且因子倍率进一步+0.5<br>受击时，若对抗失败，则当次因子倍率+0.2|该武器已印刻无羁刻痕且尚未印刻猛袭刻痕
特级|控驭刻痕|该武器已印刻奋发刻痕的话，使用该武器进攻且对抗失败时，若对方阈值高于你，则每高10点便令下次进攻检定值-7|该武器已印刻奋发刻痕且尚未印刻控驭刻痕
特级|混沌刻痕|战斗开始时，进行一次幸运检定，若成功则本场战斗该武器进攻阈值获得（差值/5四舍五入）奖励，若失败则本场战斗该武器进攻检定值获得（差值/5四舍五入）奖励；若出现大成功则固定为阈值+20，若出现大失败则固定为检定值-20|该武器尚未印刻混沌刻痕
