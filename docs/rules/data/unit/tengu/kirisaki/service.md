# 桐坂众战勤

## 形象

暂缺

## 种族（不会应用其种族特性）

鸦天狗、狐天狗、鲸天狗（最多）

## 简介

桐坂众战勤负责支援整个战线，让散兵和狙击手能够事半功倍地完成他们的使命。他们通常会练习投掷技术，将补给高效地送往散兵的手中，善于制造桐坂流专用的辅助器械，尽可能地提升狙击手的射击效率，信奉锻造大神。

## 活动范围

桐坂散兵、狙击手和战勤基本以小队的形式活动。像桐坂众这样的佣兵集团，原本在天狗国各处接活。现在业务被迫扩展到彼岸，在哪看见他们都不会奇怪了。
