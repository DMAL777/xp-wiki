# 死翼鸟

## 概述

被冠以工会名称的领导者，代表了工会的最高执行者。只有最精湛的技艺才能攀登至这顶峰。死翼鸟群决定着工会内所有的事项，只会参与最困难的争斗，其战斗与渗透能力已经是数万中无一的强大。目前没有任何的对外目击记录，对外界他们似乎只是个传说而已……………………<br>“……………………收尾工作该开始了”

## 必要条件

* 血喙、魇羽、疫心其中一职业达到6级
* 旗手达到5级

## 关键属性

* 全部

## 熟练技能

* 无
  
## 职业特性

继承血喙、魇羽、瘴骨中已有职业的全部特性与熟练技能。
