# 步兵

## 概述

步兵是联合王国的基本战力构成，是联合王国几乎所有高级军种的起点，军职一般为教士。

通过精修步兵职业，你可以学到独特的巨盾用法，获得强大的防守能力。

## 必要条件

* 信仰天教
* 拥有<a href="../../../basicJob/Guard" target="_blank">卫士</a>职业等级

## 关键属性

* 体质
* 力量

## 熟练技能

* 抵抗
* 防御
  
## 职业特性

继承卫士的职业特性；自带“顶盾”技能

!!! success "顶盾（长柄武器10级通用技能）"
    当副武器为盾系武器时，长柄武器均可视为单手武器，无需切换武器也能享受两者特殊效果带来的增益。
